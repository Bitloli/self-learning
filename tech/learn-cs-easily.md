# 面向就业学计算机
回顾大学四年，发现其中很多时间被浪费在大学物理及试验，微积分，数字电路和模拟电路上面了，但是对于一个想要从事互联方面工作的人，那些都是无所谓的东西。可能有的人会说，这些课程都是提升一个人Level的课程，
计算机里面的课程都是表面功夫，但是我觉得如果没有学会这些表面功夫，就算底蕴再厚，也是徒劳。
