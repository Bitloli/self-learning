# 在 launch 中添加一个 entry

```txt
➜ cd /home/maritns3/.local/share/applications
➜ cat sourcetrail.desktop       
[Desktop Entry]
Name=sourcetrail
Comment=source code trail
Exec=/home/maritns3/.local/bin/sourcetrail
Icon=/home/maritns3/Pictures/sourcetrail.png
Terminal=false
Type=Application
Categories=Development
```
